# Databases and SQL for Data Science with Python — IBM

This repository documents my work through **Databases and SQL for Data Science with Python** (IBM, Coursera), covering core SQL concepts and their integration with Python for data analysis.

## About Me

I'm an MSc Mathematics student transitioning into Data Science and AI. This repo is part of my structured roadmap — building SQL fundamentals alongside Python, before moving into machine learning and applied data science.

## What's Inside

- **7 SQL query files** (`.sql`): Practice queries covering schema design, data manipulation, filtering, joins, and aggregation.
- **4 Jupyter notebooks** (`.ipynb`): Hands-on work connecting SQL databases with Python — running queries, loading results into Pandas, and basic data analysis.

## Topics Covered

| Area | Concepts |
|---|---|
| Schema Operations | `CREATE TABLE`, `ALTER TABLE`, `DROP TABLE`, `TRUNCATE` |
| Data Manipulation | `INSERT`, `UPDATE`, `DELETE FROM`, `SELECT` |
| Filtering & Sorting | `WHERE`, `ORDER BY`, `LIMIT` |
| Aggregation | `GROUP BY`, `COUNT`, `SUM`, `AVG` |
| Joins | Combining data across multiple tables |
| Python-SQL Integration | Connecting to databases, running queries via Python, loading results into Pandas DataFrames |

## Why This Matters

SQL is a core skill for any Data Science/AI role — most real-world data lives in databases, not clean CSVs. This repo builds that foundation, paired with my Python skills from Python for Everybody, so I can query, extract, and analyze data end-to-end.
